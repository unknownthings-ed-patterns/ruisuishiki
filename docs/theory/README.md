# 推理式指導算術 — AI 時代の再構成

このリポジトリには、`ruisuishiki` というアプリのコードと、その背後で動いている **教育の思想を記した文書** が同居しています。アプリは消えるかもしれませんが、文書は残ってほしい——その願いから、別ライセンスで公開しています。

## このプロジェクトが目指したもの

`ruisuishiki` は、戸田城外『推理式指導算術』(1930) と『推理式指導読方』の方法論を AI 時代に再構成する試みです。

戸田が紙に焼き込んだ「**比較を基盤とする近転移エンジン**」——同じ文型の問題を、配列の妙で少しずつ変えていくことで、学習者が「自分で型を見抜く」体験をデザインする——これを現代の AI と組み合わせて、子どもも大人も使える教材生成・学習プラットフォームとして実装したものです。

## 「教育のパタン・ランゲージ」の実践として

`ruisuishiki` は単に推理式の再構成だけを目指したわけではありません。クリストファー・アレグザンダーの **パタン・ランゲージ** の発想を教育に応用した思想——

> **よい教育は、そこに働くパタンの圧縮として観察できる。**

——を背骨に置いています。

具体的には：

- **胚細胞モデル**（ユーリア・エンゲストローム）：単元の中心概念を $1$ 文の問いに絞る発想。系列の「中心の問い」はこのモデルに基づきます
- **駆動質問**（探究プロジェクトの中心問い）：単元全体を動かし続ける問い。問題画面と系列カタログの両方に常駐する `drivingQuestion` フィールドはこの実装
- **戸田の推理式**：比較を指さす AI ・ $5$ つの変化オペレータ（同・逆・$+\alpha$・質的変化・複合）・フェードイン解説 $3$ 層

これらは別個の理論ではなく、「**学習者が考える瞬間を、教育者がどう設計するか**」という $1$ つの問いに対する、複数の世代・分野からの応答です。`ruisuishiki` は、その応答たちを $1$ つのコードに圧縮した、**教育のパタン・ランゲージの実践** です。

教育のパタン・ランゲージ Wiki 本体は別サイトで公開されています：[https://unknownthings-ed-patterns.github.io/unknownthings-ed-patterns/](https://unknownthings-ed-patterns.github.io/unknownthings-ed-patterns/)
